# Algorithm Archive

꾸준한 문제 풀이를 통해 자료구조와 알고리즘 기초를 다지는 기록 저장소입니다.  
단순 풀이를 넘어, 문제 해결 과정과 사고 흐름을 정리하는 것을 목표로 합니다.

---

## 📌 Structure

algorithm-archive/
├── Season01  # 2022 ~ 2024 (초기 학습 기록)
├── Season02  # 2026 ~ (재학습 및 심화 과정)

---

## 🚀 Season Overview

### Season01
- 알고리즘 문제 풀이에 익숙해지기 위한 초기 학습 기록
- 기본적인 자료구조 및 구현 문제 중심
- 자동 업로드(BaekjoonHub) 기반 풀이 포함

### Season02
- 자료구조와 알고리즘을 더 깊이 있게 재학습
- 단순 풀이가 아닌 구현 방식과 로직 이해 중심
- 코드 구조 개선 및 Pythonic한 풀이 연습
